## Section: The problem

### Heading

AI can generate code fast. It still leaves humans with the risky part.

### Body

The real bottleneck is not getting a diff. It is answering:

- Did the AI stay inside the right scope?
- Did it actually run the right validation?
- Did it quietly make the repo harder to maintain?
- Is this safe enough to merge without reading every line?
